# 🛠 Installation

Clone the repository:

```
git clone https://github.com/Manik-0107/Python-Mini-Project.git
```

Navigate to the project folder:

```
cd Python-Mini-Project
```

Install dependencies:

```
pip install -r requirements.txt
```

Run a project:

```
python "29 Global Distance Calculator Pro.py"
```

---

# 📦 Requirements

Create a `requirements.txt` file in the repository with the following example dependencies:

```
opencv-python
numpy
matplotlib
requests
pyttsx3
pillow
```

Install them using:

```
pip install -r requirements.txt
```
